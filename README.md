### Some Features:

### Technology stack

- bcrypt
- cors
- dotenv
- jsonwebtoken
- multer
- validator
- axios
- react-toastify
- cloudinary
- stripe


`=> You can Generate JWT_SECRET by using this command : "openssl rand -base64 32", run this command on gitbash terminal`

### Setup .env file

```js

* Frontend && Admin :
VITE_BACKEND_URL = ...

* Backend :
PORT=...
MONGODB_URL=...
CLOUDINARY_API_KEY=...
CLOUDINARY_SECRET_KEY=...
CLOUDINARY_NAME=...
JWT_SECRET=...
ADMIN_EMAIL=...
ADMIN_PASSWORD=...
STRIPE_SECRET_KEY = ...



```

`Backend is Running on Port: 4000 & Frontend is Running on Port: 3000 & Admin Panel is running on Port: 3001`

```diff
=> Command For Start Server and App
```
